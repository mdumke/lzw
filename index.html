What's going on here? The compression algorithm devised by Lempel and Ziv (1977/8), with improvements by Welch (1984) lies behind formats such as zip or png. We can see how the algorithms adapts to repetitions in the content. Whenever the encoder encounters a pattern already in memory, the respective register is highlighted. For a detailed explanation of how the algorithm works, check out this blog post. For more on the topic of source coding in general, check out MIT's Digital Communication Systems.
